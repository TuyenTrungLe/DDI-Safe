# DDI-Safe

AI-powered Drug-Drug Interaction Detection System

## Overview

DDI-Safe is an application that uses AI to detect and alert users about potential drug-drug interactions, helping ensure safety when using multiple medications simultaneously.

## Project Structure

```
DDI-Safe/
├── app/
│   ├── api/          # API endpoints
│   ├── core/         # Core configuration
│   ├── models/       # Data models
│   └── agents/       # AI agents
├── backend/          # Backend services
└── docs/             # Documentation
```

## Requirements

- Python 3.8+
- pip or conda

## Installation

1. Clone repository:
```bash
git clone <repository-url>
cd DDI-Safe
```

2. Create virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# or
.\venv\Scripts\activate   # Windows
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Configure environment variables:
```bash
cp .env.example .env
# Edit .env with necessary configurations
```

## Usage

```bash
# Run the application
python main.py
```
